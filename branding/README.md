# EcoHandel Branding Assets

Centrale branding-map voor latere EcoHandel projecten.

## Hoofdlocatie
- `ecohandel/branding/logo/`

## Inhoud
Deze map is gedownload uit de gedeelde EcoHandel branding Drive en bevat o.a.:
- logo varianten
- wordmark + icon combinaties
- RGB / CMYK exports
- PNG / SVG / PDF / AI / EPS bestanden
- favicon assets (indien aanwezig in de submappen)

## Belangrijke brandingnotitie
- **Lettertype:** DM Sans

## Gebruik
Gebruik deze map als vaste bron voor:
- Econtrol Room branding
- favicon / app icon
- partner campaign assets
- prijslijst branding
- toekomstige EcoHandel UI/projecten

## Opmerking
Als nieuwe branding-assets uit Drive komen, voeg ze hier toe zodat OpenClaw en andere agents met dezelfde bron werken.
