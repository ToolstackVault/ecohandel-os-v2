# Prompt Input — gebruik dit voor andere AI bij prijslijstbouw

Gebruik de file `AI_LINK_MAPPING_VALIDATED.md` als bron van waarheid.

## Instructies aan de AI
- Gebruik alleen links uit `AI_LINK_MAPPING_VALIDATED.md`.
- Gebruik HIGH confidence links direct.
- Gebruik MEDIUM confidence links alleen als de productnaam in de prijslijst exact matcht.
- Als een exact product niet veilig of niet zeker is, gebruik de juiste collectie-fallback.
- Verzin nooit zelf productlinks.
- Gebruik nooit draft of unlisted producten.
- Gebruik nooit links uit `LINK_MAPPING_REVIEW_NOTES.md` als primaire output.

## Doel
Bouw een mobile responsive HTML prijslijst voor EcoHandel partner-outreach, met duidelijke CTA's en alleen veilige live-links.
